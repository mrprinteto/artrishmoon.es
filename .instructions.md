## ArtrishMoon Brand Rules

You are working under the ArtrishMoon brand identity. Apply the following rules to every piece of content, UI, or design you generate.

### Identity
- Brand name: ArtrishMoon
- Handle / username: @artrishmoon (all platforms)
- Brand kit: brand.artrishmoon.es
- Context: Spanish-speaking creative brand focused on lovely handmade crafts. The brand feels cute, magical, personal, artistic and warm. It is not corporate or formal. Patricia Nuñez is the founder and main creative force behind the brand.

### Color Palette
Use only these six colors. Never invent new ones.

| Name             | Hex       | Usage                                    |
|------------------|-----------|-------------------------------------------|
| Amethyst         | #ac8ec2   | Primary accent (main)                      |
| Light Amethyst   | #cdb2e0   | Secondary accent, labels, muted highlights |
| Dark Amethyst    | #8a6f9e   | Surfaces, borders, dark-theme accents      |
| Tangerine        | #f3997c   | Primary accent (main)                      |
| Light Tangerine  | #ffbea8   | Secondary accent, dark-theme highlights    |
| Dark Tangerine   | #d2795c   | Hover states, dark-theme interactions      |

Default theme is **light** (soft off-white background). A dark theme is available via toggle, using deeper amethyst/tangerine tones.

### Typography
- **Handwriting font**: WindSong — weights 400, 500. Use sparingly for decorative accents (e.g. the "moon" in the wordmark, signature names).
- **Serif font**: Noto Serif — variable weights 100–900. Use for headings and display text.
- **Sans font**: Inter — variable weights 100–900. Use for body copy, labels, and UI text.

### Logo & Assets
Logo files are served locally from `/public`:

| File                             | Path                                  | Background |
|----------------------------------|----------------------------------------|------------|
| Logo (icon + wordmark)            | /public/logo-relleno-color.png         | Light only |
| Isologo (icon only)               | /public/isologo-relleno-color.png      | Light only |
| Wordmark only                     | /public/logo-texto-relleno-color.png   | Light only |

No negative/white logo variants exist yet. Only place the logo on light or white backgrounds until a negative version is produced.

### Logo Usage Rules
DO:
- Use the logo only on light/white backgrounds where the Amethyst–Tangerine gradient reads clearly.
- Maintain a clear space around the logo equal to the height of the "A" on all sides.
- Scale the isologo to a minimum of 24 px; the horizontal logo to a minimum of 120 px wide.

DO NOT:
- Recolor the gradient, or add shadows, outlines, glows, or any external effects to the logo.
- Place the logo on dark backgrounds (no negative version exists yet).
- Stretch, distort, or rotate the logo.

### Voice & Tone
- Warm, artistic, and personal. Avoid corporate or overly formal language.
- Spanish is the primary language for content unless the project explicitly targets other markets.
